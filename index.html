Visual Studio is an IDE which lets you create apps and games using code. In order to set Visual Studio up you need to first check which device you are using. If you are using a Linux/Chromebook, Visual Studio would not work. You will have to use either a MacOS or a Windows in order to be able to use Visual Studio. On Windows, you will need to download Visual Studio by clicking on the "Type to Search" button at the bottom of your screen. Then search up Visual Studio and it will give you an option to download it. In order to download it on MacOS, you will need to download it using App Store.


The console screen is used a lot during coding. You can access this screen but double tapping on the screen that has the outcome of your code. Then click inspect, and go to the console section which could be found beside the section you are currently on. In order to print something onto the console you have to use console.log()
